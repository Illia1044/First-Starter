# Створення, виконання та запуск Java-програм 

Це перша лабораторна робота в нашому курсі з ООП, в ході якої ви маєте отримати навички написання (чи редагування), компіляції та перевірки працездатності (запуску) Java-програм. Робота дуже маленька та складається з двох завдань.

## Завдання 1. Створення та запуск першої програми
1. клонуйте створений для вас репозиторій в Netbeans (детальна інструкція з картинками є [тут](https://netbeans.org/kb/docs/ide/git.html)), відкрийте в [Repl.It](https://repl.it/) ([ось](https://repl.it/talk/learn/Configuring-GitHub-repos-to-run-on-Replit-and-contributing-back/23948) покрокова інструкція з картинками) або просто завантажте його вміст у ZIP-форматі (скориставшись кнопкою &#2193; Code" > "Download ZIP") і відкрийте в IDE за вашим вибором
